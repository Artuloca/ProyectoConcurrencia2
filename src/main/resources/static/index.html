Guía de Iniciación del Frontend

Iniciar el Frontend (Angular)

Para ejecutar el frontend, sigue estos pasos:

  • Abre la terminal en el proyecto y navega al directorio del frontend:
    • cd frontend
  • Instala las dependencias (si es la primera vez que ejecutas el frontend):
    • npm install
  • Inicia el servidor de Angular:
    • ng serve
  • El frontend debería estar corriendo en: http://localhost:4200

Acceso a la Aplicación

Asegúrate de que el frontend esté corriendo.

Abre tu navegador y visita http://localhost:4200 para ver la aplicación.
